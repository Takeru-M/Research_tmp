// 活動の流れ
1. 学習者は資料（pdfファイル）をアップロードし，意思決定をしたと思う箇所に対して，意思決定までの思考プロセスを記述する
2. 学習者は意思決定の過程において考えられていなかった他の選択肢について目をむけ，可能性を探る
3. 学習者はLLMからの示唆を受け，他の選択肢について考える
4. 学習者は現状の選択肢を踏まえて意思決定のための吟味を行う
5. 学習者はLLMからの示唆を受け，再度吟味を行う
6. ファイルをエクスポートし，活動終了

// 意思決定プロセスの記述する活動について
社会的な意思決定が行われる場に臨む前の資料は，意思決定を行うための判断材料となる情報が含まれているはずである．ここで，情報はただ単に検索等で得られる情報だけではなく，新たな提案や疑問提起などの資料作成者の意思決定が含まれていることが期待される．例えば，研究MTでは前回の振り返りや進捗の報告，新たに提案・議論したいことなどが含まれていると思われる．ここで，特に自身が意思決定を行ったと思う箇所（研究MTの文脈では主に新しく提案・議論したいこと）にハイライトをつけ，自身がどのような思考過程を経てその意思決定を行ったのかについてコメントを記述する．
